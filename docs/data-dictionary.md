# 데이터 사전

## channel 데이터

파일 위치: `data/channel/YYYY-MM-DD.parquet`

| 컬럼 | 한글명 | 타입 | 단위 | 설명 |
|------|--------|------|------|------|
| 일 | 날짜 | date | - | 데이터 집계 날짜 |
| 채널 | 채널 | str | - | 구글 / 메타 / 네이버 |
| 채널분류 | 채널 분류 | str | - | 외부(구글·메타) / 자체(네이버) |
| 캠페인 | 캠페인 코드 | str | - | `{채널코드}_CMP_{번호}_{목적}` |
| 캠페인목적 | 캠페인 목적 | str | - | 캠페인 코드의 한글 목적명 |
| 그룹 | 타겟 그룹 | str | - | 논타겟 / 유사타겟 / 리마케팅 / VIP / 윈백 |
| 소재 | 소재 코드 | str | - | `{타입}_{카테고리}_{시즌}_{AB}_{버전}` |
| 노출 | 노출 수 | int | 회 | 광고가 사용자에게 노출된 횟수 |
| 클릭 | 클릭 수 | int | 회 | 광고 클릭 수 (매체 보고 기준) |
| 비용 | 광고비 | int | 원 | 해당 소재에 집행된 광고 비용 |
| 회원가입 | 회원가입 전환 수 | int | 명 | 광고 클릭 후 회원가입 완료 수 |
| 구매 | 구매 전환 수 | int | 건 | 광고 클릭 후 구매 완료 수 |
| 구매매출 | 구매 매출 | int | 원 | 구매 전환으로 발생한 총 매출 |

## appsflyer 데이터

파일 위치: `data/appsflyer/YYYY-MM-DD.parquet`

| 컬럼 | 한글명 | 타입 | 단위 | 설명 |
|------|--------|------|------|------|
| 일 | 날짜 | date | - | 데이터 집계 날짜 |
| 미디어소스 | 미디어소스 | str | - | googleadwords_int / Facebook Ads / naver_search |
| 캠페인 | 캠페인 코드 | str | - | channel 데이터와 동일 값 |
| 그룹 | 타겟 그룹 | str | - | channel 데이터와 동일 값 |
| 소재 | 소재 코드 | str | - | channel 데이터와 동일 값 |
| 클릭 | 어트리뷰션 클릭 수 | int | 회 | 앱스플라이어 어트리뷰션 기준 클릭 |
| 회원가입 | 회원가입 전환 수 | int | 명 | 앱스플라이어 어트리뷰션 기준 |
| 구매 | 구매 전환 수 | int | 건 | 앱스플라이어 어트리뷰션 기준 |
| 구매매출 | 구매 매출 | int | 원 | 앱스플라이어 어트리뷰션 기준 |

---

## 조인 후 merged 데이터

`channel + appsflyer` 조인 결과. 분석은 이 데이터 기준.

| 컬럼 | 출처 | 비고 |
|------|------|------|
| 날짜~소재 (8개) | channel | 기준 키 |
| 노출·클릭·비용·회원가입·구매·구매매출 | channel | 주 수치 |
| 클릭_af·회원가입_af·구매_af·구매매출_af | appsflyer | 어트리뷰션 참고값 |
| CTR·CVR·CPC·CPA·ROAS | 계산 | `docs/kpi-definitions.md` 참고 |
