## 목적

매주 목요일 오후 9시, 풋웨어_소재관리 시트의 UID 목록을 읽어
무신사 상품 페이지의 현재 할인율/할인가를 크롤링하고
기존 값(V열)과 비교해 W열에 결과를 기재한다.

---

## 대상 시트

- 스프레드시트 ID: `1LGnfvS8qyILq_Lu6I_PoZGlwzJVGjKMcw_FLMq-PtwY`
- 시트명: `풋웨어_소재관리`
- U열: UID (비어있으면 스킵)
- V열: 기존 최대 할인율/할인가
- W열: 체크 결과 기재 대상

---

## 처리 흐름

```
시트 읽기 (U, V열 전체)
  └→ U열 비어있으면 스킵
  └→ UID로 무신사 상품 페이지 크롤링
       https://www.musinsa.com/products/{UID}
       └→ 품절 감지 → W열: "품절", 빨간 배경
       └→ 크롤링 실패 → W열: "실패", 노란 배경
       └→ 성공 → 할인율(%) 또는 할인가(원) 추출
            └→ V열과 동일 → W열: 현재 값, 서식 없음
            └→ V열과 다름 → W열: 현재 값, 빨간 배경 + 볼드
```

---

## W열 기재 규칙

| 상황 | W열 값 | 서식 |
|------|--------|------|
| V열과 동일 | 현재 값 (예: `81%`, `300원`) | 없음 |
| V열과 다름 | 현재 값 | 배경 빨간색 (#FF0000) + 볼드 |
| 품절 | `품절` | 배경 빨간색 (#FF0000) + 볼드 |
| 크롤링 실패 | `실패` | 배경 노란색 (#FFFF00) |

## 실행 방법

### 1. 자동 실행
- 매주 목요일 오후 21:00 Claude Code 스케줄 루틴으로 실행
- `check_discounts.py` 스크립트 호출

### 2. 수동 실행
- Streamlit 대시보드 사이드바에 **"🔍 할인가 체크 실행"** 버튼 추가
- 버튼 클릭 → `check_discounts.py` 실행 → 완료 후 결과 요약 표시
  - 예: "총 12개 체크 완료 | 변경 3건 | 품절 1건 | 실패 0건"
