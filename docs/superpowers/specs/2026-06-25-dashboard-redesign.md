# 대시보드 재설계 스펙

Date: 2026-06-25
Status: 승인됨

---

## 배경

퍼포먼스 마케팅 팀이 매일 채널·앱스플라이어 데이터를 올리고 성과를 확인한다.
현재 대시보드(채널별/캠페인별/소재별/원본)는 탭 목적이 업무 흐름과 맞지 않아
매일 아침 빠른 확인과 소재 최적화 판단이 불편하다.

---

## 목표

1. 매일 아침 30초 안에 전날 이상 없는지 확인 가능
2. 전일/전주 동요일 대비 변화율을 토글로 전환
3. 소재 A/B 비교 및 신규 소재 추이를 한 화면에서 판단
4. 캠페인 목적(퍼널 단계)별 필터로 같은 목적끼리만 비교

---

## 탭 구조

```
📋 Daily 요약 | 📊 채널·캠페인 | 🎨 소재 분석 | 🔍 원본 데이터
```

---

## 탭별 상세

### 📋 Daily 요약

**목적:** 매일 아침 전날 성과 빠른 체크

**구성:**
- 비교 기준 토글 (전일 대비 / 전주 동요일 대비) — 사이드바 또는 탭 상단
- KPI 카드 6개: 노출·클릭·비용·ROAS·구매·CPA
  - 각 카드에 어제 수치 + 변화율(%) + 방향 화살표
  - ROAS 카드는 현재처럼 배너로 강조
- 채널별 ROAS 순위 가로 바차트 (어제 기준)
- 이상치 플래그 테이블
  - `docs/data-quality-rules.md` 기준 적용
  - 이상 항목이 없으면 "✅ 이상 없음" 메시지

**데이터:** 어제 날짜 1일치만 사용 (전체 기간 필터 무시)

---

### 📊 채널·캠페인

**목적:** 예산 배분 및 캠페인 성과 판단

**구성:**
- 캠페인 목적(퍼널 단계) 필터: 상단/중단/하단/전체
- 광고비 vs ROAS 버블차트 (버블=구매매출, 평균 ROAS 기준선)
- 채널별 광고비·ROAS 바차트
- 채널별 구매매출 파이차트
- 캠페인별 집계 테이블

**참고:** 퍼널 단계 혼합 비교 금지 — 필터 선택 시 안내 문구 표시
(`docs/kpi-definitions.md` 캠페인 목적별 주요 KPI 기준)

---

### 🎨 소재 분석

**목적:** 소재 최적화 판단 (AB 비교, 신규 추이, 하위 소재 식별)

**구성 — 3개 서브섹션:**

1. **AB 비교**
   - 같은 캠페인+그룹 내 `_A_` / `_B_` 소재 자동 페어링
   - A vs B 나란히: 노출·클릭·ROAS·CVR 비교 테이블
   - 승자 강조 (ROAS 기준)

2. **소재 추이**
   - 소재 선택 → 일별 ROAS 라인차트
   - 여러 소재 동시 선택해서 비교 가능
   - x축: 날짜, y축: ROAS(%)

3. **하위 소재 목록**
   - ROAS 기준 정렬, 하위 N개 (슬라이더)
   - 중단 후보 강조 (ROAS < 목표값 또는 < 평균의 50%)
