# KPI 정의

## 계산식 & 단위

| KPI | 계산식 | 단위 | 소수점 |
|-----|--------|------|--------|
| CTR (클릭률) | 클릭 / 노출 × 100 | % | 소수점 2자리 |
| CVR (전환율) | 구매 / 클릭 × 100 | % | 소수점 2자리 |
| CPC (클릭당 비용) | 비용 / 클릭 | 원 | 정수 |
| CPA (전환당 비용) | 비용 / 구매 | 원 | 정수 |
| ROAS | 구매매출 / 비용 × 100 | % | 정수 |
| 회원가입률 | 회원가입 / 클릭 × 100 | % | 소수점 2자리 |
| CPR (회원가입당 비용) | 비용 / 회원가입 | 원 | 정수 |

> **ROAS 단위 통일: % 사용.** 300% = 3배. 보고서에서 배수 표기 금지.

## 데이터 신뢰 소스

두 소스(channel vs appsflyer) 간 수치 차이가 발생할 경우 아래 기준 적용.

| 지표 | 신뢰 소스 | 이유 |
|------|-----------|------|
| 노출 | channel | appsflyer에 없음 |
| 비용 | channel | 매체 청구 기준 |
| 클릭 | appsflyer | 어트리뷰션 클릭 기준 |
| 회원가입 | channel | 매체 보고 기준 |
| 구매 | channel | 매체 보고 기준 |
| 구매매출 | channel | 매체 보고 기준 |

차이 발생 시: channel 기준으로 주 수치 보고, appsflyer 수치는 `(AF: N)` 병기.

## 이상치 기준

| 지표 | 의심 범위 | 조치 |
|------|-----------|------|
| CTR | < 0.1% | 소재·타겟팅 문제 의심 |
| CTR (TXT 검색) | < 1% | 키워드·문구 점검 |
| CVR | > 20% | 데이터 오류 의심, 원본 확인 |
| ROAS | < 100% | 적자 캠페인, 즉시 보고 |
| ROAS | > 5000% | 소량 집행 또는 데이터 이상 의심 |
| CPA | < 100원 | 데이터 오류 의심 |
