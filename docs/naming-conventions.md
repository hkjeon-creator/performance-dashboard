# 네이밍 컨벤션

## 캠페인 코드

```
{채널코드}_CMP_{번호}_{목적}
예: GGL_CMP_01_플러스가입
```

| 채널코드 | 채널 |
|----------|------|
| GGL | 구글 |
| META | 메타 |
| NVR | 네이버 |

### 캠페인 목록 및 목적

| 캠페인 코드 | 목적 | 퍼널 단계 | 평가 KPI |
|-------------|------|-----------|----------|
| GGL_CMP_01_플러스가입 | 신규 회원 획득 | 상단 | CPA, 회원가입률 |
| GGL_CMP_02_리타겟팅 | 재방문 유도 | 중단 | ROAS, CVR |
| GGL_CMP_03_첫구매 | 첫 구매 전환 | 중단 | CPA, CVR |
| META_CMP_01_신규유저 | 신규 회원 획득 | 상단 | CPA, 회원가입률 |
| META_CMP_02_룩얼라이크 | 유사 타겟 확장 | 상단 | CPA |
| META_CMP_03_재구매 | 재구매·리텐션 | 하단 | ROAS, 매출 |
| NVR_CMP_01_브랜드KW | 브랜드 검색 방어 | 하단 | CTR, CPC |
| NVR_CMP_02_일반KW | 비브랜드 검색 획득 | 상단~중단 | CPA, CVR |
| NVR_CMP_03_재구매 | 재구매·리텐션 | 하단 | ROAS, 매출 |

> 퍼널 단계가 다른 캠페인을 ROAS 단일 지표로 비교하지 말 것.

## 그룹(타겟) 코드

| 그룹명 | 의미 | 주 사용 캠페인 |
|--------|------|----------------|
| 논타겟 | 타겟팅 없는 broad 오디언스 | 신규획득 |

| 윈백 | 이탈 고객 재활성화 | 재구매 |

## 소재 코드

```
{소재타입}_{카테고리}_{시즌}_{AB버전}_{버전번호}
예: VID_플러스멤버십_겨울_A_v1
    IMG_적립혜택_겨울_v2        ← AB 없는 단독 소재
```

### 소재 타입

| 코드 | 형식 | 주 채널 |
|------|------|---------|
| VID | 영상 | 구글·메타 |
| IMG | 이미지 | 구글·메타·네이버 |
| CRS | 캐러셀 (카드뉴스) | 구글·메타 |
| TXT | 텍스트 | 네이버 (검색) |

> TXT는 검색 광고 특성상 CTR이 다른 타입보다 높음 — 타입 간 CTR 직접 비교 금지.

### 카테고리

| 카테고리 | 메시지 방향 |
|----------|-------------|
| 플러스멤버십 | 멤버십 가입 혜택 소구 |
| 적립혜택 | 포인트·적립 혜택 소구 |
| 할인쿠폰 | 즉시 할인 소구 |
| 배송혜택 | 무료배송·빠른배송 소구 |
| 특가 | 한정 특가 소구 |
| 신상품 | 신상품 도착 소구 |

### 시즌 코드

| 코드 | 기간 (예시) |
|------|-------------|
| 겨울 | 11월~2월 |
| 봄 | 3월~5월 |
| 여름 | 6월~8월 |
| 가을 | 9월~10월 |

### AB 버전

- `_A_` / `_B_` 포함 → AB테스트 소재. **반드시 쌍으로 비교.**
- 없음 → 단독 소재. 단독 소재와 AB 소재를 동일 기준으로 비교하지 말 것.

### 버전 번호

- `v1`, `v2` … 동일 컨셉의 수정 버전.
- v2가 v1의 개선판일 경우 이전 버전과 기간 분리 후 비교.

## 파이썬 파싱 스니펫

```python
df["소재타입"] = df["소재"].str.split("_").str[0]          # VID / IMG / CRS / TXT
df["카테고리"] = df["소재"].str.split("_").str[1]
df["시즌"]    = df["소재"].str.extract(r"_(겨울|봄|여름|가을)_")
df["is_AB"]   = df["소재"].str.contains(r"_[AB]_", regex=True)
df["AB그룹"]  = df["소재"].str.extract(r"_([AB])_")        # A or B or NaN
df["버전"]    = df["소재"].str.extract(r"(v\d+)$")
```
